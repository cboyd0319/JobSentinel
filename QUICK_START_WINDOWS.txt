📋 Requirements
───────────────
✓ Windows 11 (build 22000+)
✓ Python 3.12+ installed (with "Add to PATH" checked)
✓ Internet connection
✓ 1 GB free disk space

💾 Step 1: Download
───────────────────────
Download ZIP: https://github.com/cboyd0319/JobSentinel/archive/refs/heads/main.zip
Extract to: Desktop (or anywhere you like)

🚀 Step 2: Install (2-5 minutes)
─────────────────────────────────
1. Open the JobSentinel folder
2. Right-click "bootstrap.ps1"
3. Click "Run with PowerShell"
4. Wait for it to finish (coffee break ☕)

What bootstrap.ps1 does:
→ Checks your system is compatible
→ Downloads portable Node.js (no admin needed)
→ Creates Python virtual environment
→ Installs all dependencies
→ Creates database
→ Builds frontend
→ Runs health check

⚙️ Step 3: Configure (1 minute)
───────────────────────────────
Edit these files in Notepad:

1. config/user_prefs.json
   Add your job search keywords, location, salary preferences

2. .env (optional)
   Add Slack webhook URL if you want notifications

📱 Step 4: Run (30 seconds)
────────────────────────────
1. Right-click "run.ps1"
2. Click "Run with PowerShell"
3. Open browser to: http://localhost:8000

🎉 Done! You're running JobSentinel!

═══════════════════════════════════════════════════════════════════════════════

📚 Common Commands
──────────────────

Start JobSentinel:
  .\run.ps1                    # Start API on port 8000

Start with frontend dev mode (hot reload):
  .\run.ps1 -Mode dev          # API on 8000, frontend on 3000

Run job scraper once:
  .\run.ps1 -Mode once         # Scrape jobs and exit

Test scraper without alerts:
  .\run.ps1 -Mode once -DryRun # Test mode (no Slack alerts)

Verify everything works:
  python scripts/test_stack.py # Run all tests

🔧 Troubleshooting
──────────────────

"Python not found"
  → Install Python 3.12+ from: https://www.python.org/downloads/
  → IMPORTANT: Check "Add Python to PATH" during installation

"Script cannot be loaded"
  → Right-click bootstrap.ps1 → Properties → Unblock → OK
  → Or run: powershell -ExecutionPolicy Bypass -File bootstrap.ps1

"Port 8000 already in use"
  → Close other applications using port 8000
  → Or use different port: .\run.ps1 -Port 9000

Need more help?
  → See: docs/WINDOWS_TROUBLESHOOTING.md
  → Or: docs/WINDOWS_QUICK_START.md

📂 What Gets Created
────────────────────

.tools/               Portable Node.js (no admin needed)
.venv/                Python virtual environment
data/                 SQLite database (jobs.sqlite)
static/frontend/      Built React 19 UI
.env                  Environment variables (from .env.example)
config/user_prefs.json Your job search preferences

All files stay in the JobSentinel folder. No system changes.
To uninstall: Just delete the folder. That's it!
