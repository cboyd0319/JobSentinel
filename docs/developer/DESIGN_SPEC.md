# Design Spec Location

Root [DESIGN.md](../../DESIGN.md) is the canonical design-system source for
JobSentinel.

The maintained product design spec lives at
[docs/design/design-spec.md](../design/design-spec.md). This developer path is
kept as a compatibility pointer because an early full draft was copied here.

Do not paste long design drafts into `docs/developer/`. Keep maintained design
guidance under `docs/design/` so harness checks and reviewers can keep it
current.
