# JobSentinel v2.3.0 - Advanced Resume Matching

**Release Date:** January 17, 2026
**Status:** Complete ✅

## Overview

Major enhancement to JobSentinel's resume module with intelligent matching across skills,
experience, and education. Implementation completed in 7 phases.

---

## Key Features

### Phase 1: Skill Validation UI

- **Editable Skills** - Edit, delete, and add skills from resume
- Proficiency dropdown (Beginner/Intermediate/Advanced/Expert)
- Years of experience input field
- "Add Skill" button for manual additions
- **3 new commands**: `update_user_skill`, `delete_user_skill`, `add_user_skill`

### Phase 2: Resume Library UI

- **Multiple Resume Support** - Manage multiple resume versions
- Resume dropdown in header
- Active resume indicator with upload date
- Delete button for removing old resumes
- **2 new commands**: `list_all_resumes`, `delete_resume`

### Phase 3: Experience Matching

- **Years of Experience Extraction** - Regex-based from job descriptions
- Patterns: "5+ years Python", "3-5 years experience", "Senior (7+ years)"
- Partial credit scoring: `user_years / required_years`
- Full credit (1.0) when user meets/exceeds requirement
- **New type**: `ExperienceRequirement` struct

### Phase 4: Education Matching

- **Degree Level Comparison** - Hierarchical education matching
- `DegreeLevel` enum: None(0), HighSchool(1), Associate(2), Bachelor(3), Master(4), PhD(5)
- Extracts: "Bachelor's required", "Master's preferred", "PhD in CS"
- Partial credit for lower degrees
- **New type**: `EducationRequirement` struct

### Phase 5: PDF Export

- **Browser Print-to-PDF** - Export using browser print
- Renders HTML in hidden iframe
- Works with all 5 templates
- **New command**: `export_resume_html`

### Phase 6: OCR Support (Optional)

- **Scanned PDF Parsing** - OCR fallback for image-based PDFs
- Uses `tesseract` and `pdftoppm` (command-line)
- Auto-detects when pdf-extract returns < 100 chars
- Converts to 300 DPI images
- **New Cargo feature**: `ocr` (disabled by default)

### Phase 7: Enhanced Skill Database

- **300+ Skills** across 10 categories
- **New categories**: methodology, certification, security, data
- Added: DevSecOps, OWASP, TDD, CI/CD, MLOps, Terraform, Kubernetes
- 100% self-contained - no external dependencies
- Works fully offline

---

## Weighted Match Scoring

```text
overall_score = (skills × 0.5) + (experience × 0.3) + (education × 0.2)
```

- **Skills (50%)**: 300+ keyword matching
- **Experience (30%)**: Years vs requirements
- **Education (20%)**: Degree level vs requirements
