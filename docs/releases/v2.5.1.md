# JobSentinel v2.5.1 - Production Release

**Release Date:** January 18, 2026
**Status:** Complete ✅

## Overview

First official production release with installable binaries for Windows and macOS.
This release marks JobSentinel as ready for public beta testing.

### Official Installers

Download from [GitHub Releases](https://github.com/cboyd0319/JobSentinel/releases/tag/v2.5.1).

### Code Quality Sprint

Major improvements from v2.5.0:

- **Resume → One-Click Apply Integration** — Resume file path flows from Resume Builder
- **PDF Export** — Browser-based print dialog for cross-platform PDF generation
- **HTTP Retry Logic** — Exponential backoff for transient scraper failures
- **Response Caching** — 5-minute in-memory cache reduces API load
- **WCAG 2.1 AA Accessibility** — ARIA attributes on 12 components
- **Recharts Fix** — Resolved circular dependency warnings
- **Lazy Loading** — Charts load on-demand (387KB deferred)
- **248 new component tests** — 16 test files added

### CI/CD Pipeline

- GitHub Actions workflow for Windows builds
- SQLx offline mode for CI builds without database
- Automated test runs on push to main
