# v2.5.3 Release Notes

> Release Date: 2026-01-24
> Type: Historical feature release
> Current source-policy note: this release history has been superseded by the
> current JobSentinel source policy. LinkedIn is now a user-opened search-link
> destination only.

## Overview

JobSentinel v2.5.3 shipped an earlier LinkedIn connection experiment. That
experiment is no longer part of the active product direction.

Current behavior:

- LinkedIn opens through Deep Links in the user's browser.
- JobSentinel does not collect LinkedIn session credentials.
- JobSentinel does not monitor LinkedIn in the background.
- Automatic monitoring should use official company pages and public ATS sources
  such as Greenhouse, Lever, Ashby, SmartRecruiters, and USAJobs.

See [Job Source Adapters](../features/scrapers.md) for the active source model.
