# JobSentinel v2.9.0 Release Notes

Status: local release candidate. Public `2.9.0` assets are not published until
final target-platform builds, upload, and public verification close.

JobSentinel `2.9.0` is a privacy-first job-search readiness release. It focuses
on helping non-technical job seekers find, review, track, tailor, and follow up
on roles while keeping sensitive job-search data local by default.

## Before Installing Or Updating

- Use the official GitHub Releases page for downloads.
- Back up Settings before replacing the app.
- Verify the matching `.sha256` checksum before opening a package or installer.
- Expect `_unsigned` in Windows installer filenames when Windows signing
  credentials are unavailable.
- Expect `_no-account_` and `_universal` in macOS DMG filenames until Developer
  ID signing and notarization are available.
- JobSentinel does not silently update itself in `2.9.0`.

## What Is New

- Downloadable Agent Skills archives for job search planning, posting-risk
  review, resume tailoring, application review, tracking, outreach, interview
  prep, and offer/pay review. Release assets include both tar.gz and ZIP formats
  with matching checksums.
- Browser Import and LinkedIn-compatible user-opened workflows stay
  review-first. JobSentinel does not store LinkedIn cookies, tokens, browser
  storage, authorization headers, or offline sessions.
- Source discovery favors official feeds, public employer pages, public
  hiring-platform feeds, and user-opened restricted-source paths with clear
  warnings.
- First-run setup adds broader non-technical career paths, editable search
  suggestions, optional source choices, desktop alerts, pay-floor review, and
  freshness controls.
- Resume Builder and Resume Match add evidence-bounded bullet structures,
  readable export checks, major Greenhouse, Workday, and Taleo / Oracle
  Recruiting portal review prompts, hard-requirement review, and local
  readability guidance.
- Application Assist keeps final submission under the user's control and adds
  stronger review for repeated profile details, screening answers, attachments,
  and pay-history questions.
- Pay Protection adds written-vs-verbal offer separation, total-compensation
  review, commute and relocation prompts, deadline pressure checks, and
  counter/decline starter notes.
- Settings backup and restore now cover settings, saved searches, and cover
  letter templates. Backups include plain recovery guidance and leave saved
  connection details, passwords, tokens, cookies, browser sessions, local
  database records, and safe support reports out.
- Safe support reports can be copied or saved from recovery and support flows
  so users can review sanitized details before sharing.

## Privacy And Security

- Core data stays local by default: saved jobs, searches, notes, applications,
  resumes, salary floors, and support reports.
- External AI is optional, disabled by default, preview-gated, and routed
  through the privacy-first AI gateway.
- Restricted-source actions require explicit user choice and must not run as
  hidden background access.
- Saved secrets use the local encrypted vault path and operating-system
  credential protections where available.
- Release workflows produce checksums, SBOMs, and provenance/SBOM attestations
  for public assets.

## Platform Notes

- Windows 11+ is a primary target. Signed installers are used when credentials
  exist; no-account builds are clearly labeled `_unsigned`.
- macOS is a primary target. The no-account universal DMG is not Developer ID
  signed or notarized until Apple Developer Program materials exist, so macOS
  may require a first-open Privacy & Security approval.
- Linux packages are part of the cross-platform release path.
- Agent Skills ZIP archives are provided for the most portable Windows
  extraction path; tar.gz remains available for macOS and Linux users.

## Known Release Boundaries

- Do not publish `v2.9.0` until the final roadmap gates, target-platform
  package verification, public artifact checks, and manual UI validation are
  complete.
- macOS cannot honestly claim zero-friction Gatekeeper readiness without
  Developer ID signing, notarization, stapling, and signed-artifact
  verification.
- JobSentinel is not an application-submission bot, legal adviser,
  compensation authority, or employer-side hiring system.
