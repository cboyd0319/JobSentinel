# v2.6.0 UX Improvements

Status: Complete on main  
Released: January 24, 2026  
Version: v2.6.0

## Outcome

v2.6.0 shipped frontend UX polish across error recovery, loading states,
validation, accessibility, and render performance.

## Completed Work

- Error recovery actions in market, cover letter, analytics, and apply flows.
- Stale data indicators in dashboard and market surfaces.
- Skeleton loading states for resume and dashboard surfaces.
- Timeout feedback for longer-running company research and profile forms.
- Inline validation for interview scheduling and resume builder steps.
- Unsaved-change confirmation for cover letter template editing.
- Memoization and lookup-table cleanup for frequently rendered UI components.
- Accessible labels for resume builder and optimizer textareas.
- Optimistic update rollback for notification preferences.
- Loading-state helper patterns. The unreferenced `useMinimumLoadingDuration`
  hook was later retired from `main`.
- More actionable empty states for resume skills, experience, and education.
